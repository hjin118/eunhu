# P학원 회원 관리 시스템

학원 운영을 위한 웹 기반 회원 관리 시스템입니다. 회원 등록부터 출결, 수강 관리, 결제, 상담까지 원스톱으로 관리할 수 있습니다.

## 주요 기능

### 회원 관리
- 회원 등록/수정/삭제
- 이름, 연락처, 학교, 학년 정보 관리
- 수강반 배정
- 검색 기능

### 출결 관리
- 날짜별 출결 기록 (출석/결석/지각)
- 오늘 출결 현황 대시보드
- 출결 기록 필터링

### 수강반 관리
- 수강반 생성/삭제
- 과목, 강사, 시간대 설정
- 회원 배정 관리

### 수업 진도
- 반별 진도 기록
- 날짜별 진도 내용 저장

### 결제 관리
- 월 회비, 교재비, 기타 비용 기록
- 결제 완료/미납 상태 관리
- 미납 회원 필터링

### 상담 관리
- 상담 유형별 기록 (입학, 진학, 성적, 행동, 기타)
- 상담 내용 및 후속 조치 저장

### 대시보드
- 총 회원 수
- 평균 출석률
- 미납 회원 현황
- 월간 수입
- 반별 인원 차트
- 월간 출결 현황 차트

### 백업/복구
- JSON 파일로 데이터 내보내기
- 데이터 파일로 복구
- 데이터 초기화 기능

## 시작하기

### 설치

1. 이 저장소를 클론합니다:
```bash
git clone https://github.com/hjin118/eunhu.git
cd eunhu
```

2. 웹 브라우저에서 `index.html` 파일을 엽니다.

### 사용법

1. **회원 등록**: 회원 관리 탭에서 이름, 연락처 등 기본 정보를 입력하고 등록합니다.
2. **수강반 생성**: 수강반 관리 탭에서 반을 생성하고 회원을 배정합니다.
3. **출결 기록**: 출결 관리 탭에서 날짜와 회원을 선택하고 출결 상태를 기록합니다.
4. **결제 관리**: 결제 관리 탭에서 회원의 결제 내역을 등록하고 관리합니다.

## 기술 스택

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **데이터 저장**: LocalStorage
- **차트**: Chart.js (대시보드)
- **호환성**: Chrome, Edge, Firefox, Safari (최신 버전)

## 데이터 구조

데이터는 브라우저의 LocalStorage에 저장됩니다:

- `eunhu_members`: 회원 목록
- `eunhu_classes`: 수강반 목록
- `eunhu_backup_logs`: 백업/복구 기록

## 파일 구조

```
eunhu/
├── index.html      # 메인 HTML 파일
├── style.css       # 스타일시트
├── script.js       # 자바스크립트 로직

├── styles.css      # 추가 스타일 (사용하지 않음)
├── PRD.md          # 제품 요구사항 문서

```
